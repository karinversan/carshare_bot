# Data

## Public bootstrap datasets
- CarDD
- TQVCD
- CrashCar101 (augmentation source only)

## Important caveat
These are research bootstrap datasets and do not substitute for product-like paired inspection data.

## Custom product-like protocol
- 20 to 30 cars
- 2 inspections per car
- 4 required views per inspection
- optional close-ups
- annotations:
  - viewpoint
  - quality acceptance/rejection
  - damage type
  - bbox or mask
  - pre-existing vs new
  - review status
